syzoj 密码错误

使用 node app.js 启动网页端发生如下错误：

原因是本地数据库密码和 config.json 配置文件的密码不一致，请仔细检查 sql 命令中的密码和配置文件的密码是否一致，即

/opt/syzoj/web/config.json 文件中 password 字段是否一致。
