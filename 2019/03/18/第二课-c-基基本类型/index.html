c++ 基本数据类型

char 字符类型

int 整数类型

4个字节的大小，支持四则运算（加减乘除）和取余（模）。

代码：

sizeof(int); //计算int的大小

10%3; //10除以3，商3余1，所以运算结果为1

(1+2)*3 //添加小括号提高加号的优先级

double 双精度浮点数（ float单精度浮点数）

8个字节的大小，支持四则运算（加减乘除），但不支持取模。

sizeof（double） //计算double的大小
